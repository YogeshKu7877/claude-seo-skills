<!-- Ahrefs MCP tool names verified in Phase 1, prefix confirmed: mcp__claude_ai_ahrefs__ -->

Tool: mcp__claude_ai_ahrefs__site-explorer-metrics
  Param: target (bare domain, e.g., "example.com")
  Returns: organic_traffic, traffic_cost (CENTS), organic_keywords, backlinks, referring_domains

Tool: mcp__claude_ai_ahrefs__site-explorer-domain-rating
  Param: target
  Returns: domain_rating (0-100), ahrefs_rank

Tool: mcp__claude_ai_ahrefs__site-explorer-all-backlinks
  Param: target
  Returns: url_from, url_to, anchor, domain_rating_source, first_seen, lost_date

Tool: mcp__claude_ai_ahrefs__site-explorer-organic-keywords
  Param: target
  Returns: keyword, position, volume, cpc (CENTS), traffic, traffic_percent

Tool: mcp__claude_ai_ahrefs__site-explorer-organic-competitors
  Param: target
  Returns: competitor domains with keyword overlap count, shared keywords, unique keywords

Tool: mcp__claude_ai_ahrefs__keywords-explorer-matching-terms
  Param: target (keyword), possibly filter params (UNVERIFIED — must discover at runtime)
  Returns: matching keyword list with volume, difficulty, CPC
  NOTE: content-gap use case — exact filter for "competitor has, target doesn't" is unverified. Must use ToolSearch to inspect live schema before calling.

Tool: mcp__claude_ai_ahrefs__site-explorer-broken-backlinks
  Param: target
  Returns: broken backlinks with url_from, url_to, anchor, http_code, first_seen

Tool: mcp__claude_ai_ahrefs__site-explorer-referring-domains
  Param: target (+ date filter for new links)
  Returns: referring domains with domain_rating, backlinks count, first_seen

Tool: mcp__claude_ai_ahrefs__site-explorer-anchors
  Param: target
  Returns: anchor text, referring_domains count, backlinks count per anchor

Tool: mcp__claude_ai_ahrefs__site-explorer-domain-rating-history
  Param: target
  Returns: date, domain_rating, ahrefs_rank per data point

Tool: mcp__claude_ai_ahrefs__site-explorer-top-pages
  Param: target
  Returns: url, traffic, traffic_percent, keywords, top_keyword

CRITICAL CONVENTION: All monetary values (traffic_cost, cpc, any _cost or _value field) are in CENTS.
Divide by 100 before display. Format as "$X,XXX" with comma separators.

MCP Check Pattern (from mcp-degradation.md):
1. Use ToolSearch with query "+ahrefs"
2. If tools returned → Ahrefs MCP is available, proceed
3. If no tools → display Ahrefs error template from mcp-degradation.md, stop
